# Edén Nails Bar & Makeup

Página oficial administrable con Decap CMS + Netlify.

## Panel administrativo
Cuando esté conectado con Netlify y Git Gateway:
`/admin`

Desde ahí podrás editar:
- Portada
- Servicios y precios
- Promociones
- Galería
- Videos MP4
- Reglas del programa de referidas

## Formularios privados
Netlify guardará:
- `referidas-eden`
- `uso-codigo-eden`

Se ven en Netlify → Forms/Formularios.

## Importante
Para activar `/admin`:
1. Conectar este repositorio a Netlify.
2. Activar Netlify Identity.
3. Activar Git Gateway.
4. Invitar tu correo como usuaria administradora.
