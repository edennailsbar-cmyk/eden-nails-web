EDÉN NAILS - REFERIDOS CONTROLADOS POR LA DUEÑA

Cambios:
- Se quitó de la página pública la opción de crear código de referida.
- Las clientas públicas solo pueden registrar/usar un código que ya recibieron.
- Se agregó una página privada para crear códigos:
  /admin/referidos.html

Contraseña inicial:
eden2026

Dónde se guardan:
Netlify → Proyecto → Formularios → referidas-eden

Dónde ves quién usó código:
Netlify → Proyecto → Formularios → uso-codigo-eden

Flujo recomendado:
1. Tú creas el código desde /admin/referidos.html para una clienta real.
2. La clienta comparte ese código con su amiga.
3. La amiga entra a la página pública y usa el código.
4. Tú ves ambos registros en Netlify Forms.
